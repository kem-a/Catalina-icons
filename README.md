# Catalina-icons

An icon theme for Linux inspired by macOS Catalina. Available in two variants: **Catalina-dark** and **Catalina-light**.

## Sizes

| Size     | Contexts                                |
| -------- | --------------------------------------- |
| 16×16    | actions, devices, panel, places, status |
| 22×22    | actions, apps, panel, places            |
| 24×24    | actions, panel, places                  |
| 64×64    | apps                                    |
| 128×128  | apps, devices, mimetypes, places        |

All sizes are also available in HiDPI (`@2x`).

## Installation

### Manual

```bash
# For current user
cp -r Catalina-dark Catalina-light ~/.local/share/icons/

# System-wide
sudo cp -r Catalina-dark Catalina-light /usr/share/icons/
```

### Apply the theme

Use your desktop environment's settings or a tool like **GNOME Tweaks**, **LXAppearance**, or **kvantum** to select `Catalina-dark` or `Catalina-light`.
