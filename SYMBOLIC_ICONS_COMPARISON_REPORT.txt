EXECUTIVE SUMMARY
=================
The Catalina-dark and Catalina-light icon themes have significant structural and
content discrepancies in symbolic icon directories. Key findings:

1. Dark theme has EMPTY status/devices directories (no files)
2. Light theme has 600+ additional -symbolic variants in actions
3. 16x16/panel: 612+ files are byte-identical (MAJOR ISSUE)
4. Dark theme has 60-79 extra legacy/deprecated icon names
5. ColorScheme colors differ correctly where files differ

================================================================================
SUMMARY OF KEY ISSUES
================================================================================

ISSUE #1: EXCESSIVE IDENTICAL FILES IN 16x16/PANEL (CRITICAL)
  Severity: HIGH
  Impact: ~76% of common panel icons are byte-identical between themes
  Consequence: These icons will have the same appearance in both dark and
              light desktop themes, defeating the purpose of theme variants
  Affected: 16x16/panel, 16x16@2x/panel, scalable/panel (612 files each)
  Examples: All battery icons, audio volume icons, bluetooth icons, etc.
  
  Recommended Action: These 612 files need color schema updates.
  Dark theme should use lighter colors (#eff0f1 or similar)
  Light theme should use darker colors (#31363b or similar)

ISSUE #2: INCOMPLETE SYMBOLIC ICON SET IN DARK THEME
  Severity: MEDIUM
  Impact: Dark theme lacks 619 -symbolic icon variants in 16x16/actions
  Consequence: Applications expecting modern -symbolic icon names won't find
              them in dark theme; fallback behavior unpredictable
  Affected: 16x16/actions, 16x16@2x/actions, scalable/actions
  Missing pattern: *-symbolic.svg, *-symbolic-rtl.svg, etc.

ISSUE #3: LEGACY ICON ALIASES IN DARK THEME
  Severity: LOW-MEDIUM
  Impact: Dark theme has 60+ extra obsolete icon names (GPM, XFCE specific)
  Consequence: Clutters dark theme with deprecated power manager icons
  Affected: 16x16/panel (60 extra files), 24x24/panel (79 extra files)
  Examples: gpm-primary-*.svg, xfpm-*.svg, ac-adapter.svg variations
  
  Recommended Action: Consider if these are truly needed or should be
                     pruned for theme consistency

ISSUE #4: PANEL ICONS MISSING FROM LIGHT THEME
  Severity: LOW
  Impact: Light theme has fewer panel icons in some sizes
  Affected: 16x16/panel, 24x24/panel
  Gap: ~60 network/signal icons in 24x24/panel, missing from light theme
  Consequence: Potential rendering issues with network/status indicators

ISSUE #5: INCONSISTENT ACTIONS IN SYMBOLIC VARIANTS
  Severity: MEDIUM
  Impact: Asymmetry in -symbolic icon coverage
  Dark theme: Limited -symbolic variants
  Light theme: 619+ -symbolic variants
  Consequence: Inconsistent icon availability between themes
